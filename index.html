Open Source Projects Made In 2024

Here are some of the open-source projects I've made in 2024. All the code is licensed under the most permissive license out there — the MIT License (or sometimes even under the public domain). Feel free to use the code in your own projects no matter what you're doing.

  • The Tiny Lua Compiler - The smallest and easiest Lua compiler in the world, with detailed comments explaining how most compilers work. This project is perfect for beginners who want to understand the basics of compiler construction. Written in Lua without any dependencies. Licensed under the public domain.

  • undetected_geckodriver - A Python library that patches the Firefox binary to make it undetectable by websites when used with Selenium. Written in Python.

  • MathParser.lua - An advanced and fast parser and evaluator for mathematical expressions written in Lua, supporting custom functions, changing operator precedence, and more. Written in Lua.

  • LuaΞΕΝ - An archive of the entire mono-repository of the LuaXen project, including a Lua compiler, Lua VM, AST executor (interpreter), AST-based Obfuscator, VM-based Obfuscator, Prettier, Minifier, Optimizer, Packer, Assembler, and more. All wrapped up in a nice API. Written in pure Lua.

  • Github Desktop Notifier - A simple Python script that sends a Linux desktop notification whenever you get a notification on Github. This script helps developers stay updated with their GitHub notifications without constantly checking the website. Written in Python.

  • Lua.js - A library that allows you to tokenize, parse, compile, and execute Lua 5.1 - 5.4 code. Provides a nice API for further manipulation of the AST/bytecode, enabling you to write your own Lua transpiler, obfuscator, minifier, and more. Written in TypeScript.
